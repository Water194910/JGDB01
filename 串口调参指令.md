# 串口调参指令手册

## 概述

串口1 (PA9/PA10, 115200) 用于运行时参数调整。

**基本格式：**
- 命令格式：`参数名数值`（如 `XP1.5`）
- 以 `\r\n` 结尾
- 输入 `PID` 查询所有参数
- 输入 `SAVE` 保存参数到Flash

## 七、参数说明

### 7.1 位置环参数

- **KP**：比例系数，决定响应速度
  - 太小：响应慢，跟踪滞后
  - 太大：超调，振荡

- **KI**：积分系数，消除稳态误差
  - 通常设为0，避免积分饱和

- **KD**：微分系数，抑制超调
  - 通常设为0，避免噪声放大

### 7.2 速度环参数

- **KP**：比例系数，决定响应速度
- **KI**：积分系数，消除稳态误差
- **KD**：微分系数，抑制超调

### 7.3 Boost参数

- **BT**：boost启动阈值（度）
  - 位置误差超过此值时启动boost

- **BG**：boost增益系数
  - 每度误差增加的boost倍数

- **BM**：boost上限
  - boost倍数的最大值

- **BP**：boost像素阈值
  - 视觉偏差超过此像素数时启动boost

- **BD**：boost时X轴单次最大角度增量（度）
  - boost模式下每次更新的最大角度变化

- **BE**：boost时pos_err保护门限（度）
  - boost模式下位置误差超过此值时停止更新

### 7.4 IMU参数

- **KG**：IMU阻尼系数
  - 0：无阻尼
  - 0.5：中等阻尼（推荐起点）
  - 1.0：强阻尼

- **IO**：Yaw开机偏移
  - 仅用于调试，不影响控制

- **IR**：Roll开机偏移
  - Y轴IMU外环的零点

- **IU**：Y轴位置环模式
  - 0：编码器位置环（默认）
  - 1：IMU Roll姿态外环

## 八、调参流程

### 8.1 位置环调参

1. 先调X轴位置环：
   - 设置 `XI0` `XD0`（关闭积分和微分）
   - 逐步增大 `XP`，直到云台能跟踪目标
   - 如果超调，减小 `XP`

2. 再调Y轴位置环：
   - 同样方法调 `YP`

### 8.2 速度环调参

1. 设置 `XSI0` `XSD0`（关闭积分和微分）
2. 逐步增大 `XSP`，直到响应平滑
3. 如果振荡，减小 `XSP`

### 8.3 Boost调参

1. 设置 `BT10`（10度阈值）
2. 设置 `BG0.05`（0.05增益）
3. 设置 `BM3.0`（3倍上限）
4. 测试大偏差时的加速效果

### 8.4 IMU阻尼调参

1. 设置 `KG0`（关闭阻尼）
2. 手动晃动云台，观察振荡
3. 逐步增大 `KG`（0.1步长）
4. 找到振荡1-2次就稳定的值

### 8.5 Y轴IMU外环测试

1. 设置 `IU0`（编码器模式）
2. 瞄准目标
3. 设置 `IU1`（IMU模式）
4. 观察俯仰稳定性

## 九、常见问题

### Q1：参数修改后无效？

A1：检查是否保存到Flash：
```
SAVE
```

### Q2：云台振荡？

A2：减小位置环KP或速度环KP：
```
XP0.3
XSP30
```

### Q3：云台响应慢？

A3：增大位置环KP或速度环KP：
```
XP0.8
XSP80
```

### Q4：阻尼项导致抖动？

A4：减小阻尼系数或添加滤波：
```
KG0.3
```

### Q5：Y轴IMU外环不稳定？

A5：切回编码器模式：
```
IU0
```

## 十二、注意事项

1. **参数保存**：修改参数后记得 `SAVE`，否则重启丢失
2. **调参顺序**：先位置环，后速度环，最后Boost和IMU
3. **安全范围**：不要设置过大参数，避免电机过载
4. **实时观察**：调参时观察云台状态，避免振荡
5. **记录参数**：找到最佳参数后记录下来
