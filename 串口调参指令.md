# 串口调参指令手册

## 概述

串口1 (PA9/PA10, 115200) 用于运行时参数调整。

**基本格式：**
- 命令格式：`参数名数值`（如 `XP1.5`）
- 以 `\r\n` 结尾
- 输入 `PID` 查询所有参数
- 输入 `SAVE` 保存参数到Flash

## 七、参数说明

### 7.1 位置环参数

- **KP**：比例系数，决定响应速度
  - 太小：响应慢，跟踪滞后
  - 太大：超调，振荡

- **KI**：积分系数，消除稳态误差
  - 通常设为0，避免积分饱和

- **KD**：微分系数，抑制超调
  - 通常设为0，避免噪声放大

### 7.2 视觉目标比例

- **XTP / YTP**：摄像头像素偏差转换成目标角度增量的比例
  - 计算关系：`目标角度增量 = 像素偏差 * TP`
  - 太小：目标更新慢，追踪滞后
  - 太大：目标跳变大，容易抖动或过冲
  - 注意：`XP/YP` 是位置环力度，`XTP/YTP` 是视觉给目标角的步长，两者不要混在一起调

- **XSG**：X轴软响应区增益
  - 只影响 `3px < abs(dx) <= 30px` 的中心附近区域
  - 计算关系：`目标角度增量 = dx * XTP * XSG`
  - `XSG0`：软响应区不动，效果接近小死区
  - `XSG1`：软响应区使用完整 `XTP`
  - 中心附近滞后可增大到 `0.45` 或 `0.55`，中心附近摆动则减小到 `0.25`

- **XDB**：X轴/视觉调试输出
  - `XDB1`：每100ms输出 `dx/dy、X角度、目标、速度目标、位置误差、rpm、电压、XTP/XSG/KG`
  - `XDB0`：关闭输出
  - 仅用于调试，不保存到Flash

### 7.3 速度环参数

- **KP**：比例系数，决定响应速度
- **KI**：积分系数，消除稳态误差
- **KD**：微分系数，抑制超调

### 7.4 Boost参数

- **BG**：boost增益系数
  - 每度误差增加的boost倍数
  - 当前Boost倍率不再乘到X轴速度目标，默认只保留为备用参数

- **BM**：boost上限
  - boost倍数的最大值
  - 建议设置 `BM1.0` 或 `BG0`，避免大角度转向生硬和超调

- **BP**：boost像素阈值
  - 视觉偏差超过此像素数时启动boost

- **BD**：boost时X轴单次最大角度增量（度）
  - boost模式下每次更新的最大角度变化

- **BE**：boost时pos_err保护门限（度）
  - boost模式下位置误差超过此值时停止更新

### 7.5 IMU参数

- **KG**：X轴IMU角速度前馈强度
  - 0：关闭IMU前馈
  - 0.3：轻补偿
  - 0.6：更强补偿
  - 仅在视觉追踪或短暂丢帧保持时生效，不依赖视觉死区
  - 方向固定在代码里；若实测方向反，改代码中的前馈符号，不建议长期用负KG

- **IO**：Yaw开机偏移
  - 仅用于调试，不影响控制

- **IR**：Roll开机偏移
  - Y轴IMU外环的零点
  - 默认由Y轴回零流程自动设置

- **IU**：Y轴位置环模式
  - 0：编码器位置环
  - 1：IMU Roll姿态外环（默认）

- **YH**：Y轴Roll=0回零
  - 以BNO085原始Roll=0作为物理零点
  - 开机自动执行，也可运行时手动发送 `YH`
  - 回零成功后当前Roll被设为Y轴控制零点

- **YM / YT**：Y轴手动测试
  - `YM1`：视觉不再改Y目标，只测试Y轴本体闭环
  - `YM0`：恢复视觉控制Y目标
  - `YT5` / `YT-5`：手动设置Y轴目标角

## 八、调参流程

### 8.1 位置环调参

1. 先调X轴位置环：
   - 设置 `XI0` `XD0`（关闭积分和微分）
   - 设置 `XTP0.022`、`XSG0.35`（先固定视觉比例和软区增益）
   - 逐步增大 `XP`，直到云台能跟踪目标
   - 如果超调，减小 `XP`

2. 再调Y轴位置环：
   - 开机等待 `Y Roll Homing OK`
   - 如需重新回零，发送 `YH`
   - 设置 `YTP0.02` 或 `YTP0.03`
   - 同样方法调 `YP`

### 8.2 速度环调参

1. 设置 `XSI0` `XSD0`（关闭积分和微分）
2. 逐步增大 `XSP`，直到响应平滑
3. 如果振荡，减小 `XSP`

### 8.3 Boost备用设置

1. 默认建议 `BG0` 或 `BM1.0`
2. 当前代码不再把Boost倍率乘到X轴速度目标
3. `BG/BM/BP/BD/BE` 保留为备用参数，优先使用 `KG` 做动态补偿

### 8.4 X轴IMU前馈调参

1. 设置 `KG0`（关闭IMU前馈）
2. 设置 `BG0` 或 `BM1.0`（关闭Boost倍率影响）
3. 视觉锁定目标后设置 `KG0.3`
4. 轻微左右转动车体，观察目标偏差是否减小
5. 响应慢则增大 `KG`，目标附近摆动则减小 `KG`
6. 若目标偏差变大，说明前馈方向反了，需要改代码里的固定符号

### 8.5 Y轴IMU外环测试

1. 设置 `KG0`（先关闭阻尼）
2. 设置 `YM1`（Y轴手动目标模式）
3. 设置 `YP0.3`
4. 发送 `YT5`，观察Y轴是否靠近+5度
5. 发送 `YT-5`，观察Y轴是否靠近-5度
6. 确认稳定后发送 `YM0`，恢复视觉控制

## 九、常见问题

### Q1：参数修改后无效？

A1：检查是否保存到Flash：
```
SAVE
```

### Q2：云台振荡？

A2：减小位置环KP或速度环KP：
```
XP0.3
XSP30
```

### Q3：云台响应慢？

A3：优先增大视觉目标比例或软区增益，再考虑位置环KP：
```
XTP0.024
XSG0.45
XP1.2
```

### Q4：IMU前馈导致抖动？

A4：减小 `KG`：
```
KG0.3
```

### Q5：Y轴IMU外环不稳定？

A5：切回编码器模式：
```
IU0
```

## 十二、注意事项

1. **参数保存**：修改参数后记得 `SAVE`，否则重启丢失
2. **调参顺序**：先位置环，后速度环，最后IMU前馈；Boost默认不建议作为主要追赶手段
3. **安全范围**：不要设置过大参数，避免电机过载
4. **实时观察**：调参时观察云台状态，避免振荡
5. **记录参数**：找到最佳参数后记录下来
6. **Y轴回零**：开机会自动移动到BNO085原始Roll=0，若超时会输出 `Y Roll Homing Timeout`
